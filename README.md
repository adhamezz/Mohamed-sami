## ✨ الميزات

### 🎨 التصميم والواجهة
- **RTL Support**: دعم كامل للعربية والكتابة من اليمين لليسار
- **Responsive Design**: تصميم متجاوب مع جميع أحجام الشاشات
- **Modern UI**: واجهة مستخدم عصرية وأنيقة بألوان احترافية (أزرق داكن وذهبي)
- **Tailwind CSS**: نظام تنسيق قوي وقابل للتخصيص

### 🎬 الحركات والتأثيرات
- **Framer Motion**: حركات سلسة واحترافية
- **Smooth Animations**: تأثيرات الدخول والخروج والتحويم
- **Scroll Effects**: تأثيرات عند التمرير بسلاسة في الصفحة

### 📱 الأقسام الرئيسية
1. **Navbar**: شريط علوي متقدم بمعلومات التواصل + شريط التنقل الرئيسي
2. **Hero Section**: قسم البطل مع صور وأزرار CTA
3. **About**: نبذة عن المكتب مع الخصائص والمميزات
4. **Services**: الخدمات المتخصصة بكارتات تفاعلية
5. **Media Center**: مركز إعلامي بالفيديوهات والمقالات
6. **News**: أحدث الأخبار والتحديثات
7. **Legal Articles**: مقالات قانونية متخصصة
8. **Legal Library**: مكتبة قانونية برابط للتحميل
9. **Contact**: نموذج التواصل وسهولة الوصول
10. **WhatsApp Button**: زر واتساب عائم مع إشعارات
11. **Footer**: تذييل احترافي غني بالمعلومات

## 🚀 التثبيت والتشغيل

### الخطوة 1: تثبيت المتطلبات

انسخ الأمر التالي في موجه الأوامر:

``bash
npm install
```

هذا سيثبت جميع المتطلبات:
- React 19
- React Router DOM
- Framer Motion
- Tailwind CSS
- PostCSS
- Autoprefixer
- Lucide React (Icons)

### الخطوة 2: تشغيل خادم التطوير

```bash
npm run dev
```

سيتم فتح الموقع على العنوان:
```
http://localhost:5173
```

### الخطوة 3: بناء الموقع للإنتاج

```bash
npm run build
```

سيتم إنشاء ملف التجميع في مجلد `dist/`

### الخطوة 4: معاينة النسخة النهائية

```bash
npm run preview
```

## 🎨 الإعدادات والتخصيص

### تغيير الألوان

عدّل الألوان الأساسية في [tailwind.config.js](tailwind.config.js):

```javascript
colors: {
  gold: { /* اللون الذهبي */ },
  navy: { /* اللون الأزرق الداكن */ },
}
```

### تغيير الخطوط

الخطوط المستخدمة:
- **Cairo**: للعناوين الرئيسية
- **Tajawal**: للنصوص الأخرى

يمكنك تغييرها في [index.html](index.html) ضمن رابط Google Fonts

### تعديل محتوى الصفحات

كل قسم له ملفه الخاص في `/src/components/`. ما عليك سوى فتح الملف والتعديل على المحتوى والبيانات.

### استبدال أرقام الاتصال والعناوين

- البحث عن الأرقام والعناوين في ملفات المكونات
- استبدلها برقمك والعنوان الخاص بك
- البحث عن `+965 2216 1111` للعثور على جميع الأرقام

## 🌐 نشر الموقع

### Netlify (الخيار الأسهل)

1. قم بدفع المجلد إلى GitHub
2. اذهب إلى [Netlify](https://netlify.com)
3. اختر "New site from Git"
4. اتبع التعليمات

### Vercel

1. قم بدفع المجلد إلى GitHub
2. اذهب إلى [Vercel](https://vercel.com)
3. انقر على "Import Project"
4. اتبع التعليمات

### خادم تقليدي

```bash
npm run build
# ثم قم برفع محتوى المجلد dist/ إلى الخادم الخاص بك
```

## 📞 الدعم والمساعدة

### المشاكل الشائعة

**المشكلة**: الخطوط العربية لا تظهر
- **الحل**: تأكد من أن خطوط Google Fonts محملة في `index.html`
