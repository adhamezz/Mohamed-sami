# 🚀 دليل التثبيت السريع

## خطوات التثبيت والتشغيل

### 1️⃣ تثبيت المتطلبات

```powershell
# افتح موجه الأوامر في مجلد المشروع
cd c:\Users\HP\Desktop\law

# ثبت جميع الحزم المطلوبة
npm install
```

**المدة المتوقعة**: 3-5 دقائق

---

### 2️⃣ تشغيل الموقع على خادم التطوير

```powershell
npm run dev
```

**النتيجة**: سيظهر عنوان مثل:
```
Local:   http://localhost:5173
```

افتح هذا الرابط في متصفحك! 🎉

---

### 3️⃣ بناء الموقع للإنتاج

```powershell
npm run build
```

سيتم إنشاء ملفات محسّنة في مجلد `dist/`

---

### 4️⃣ معاينة النسخة النهائية

```powershell
npm run preview
```

## 🔧 المشاكل الشائعة

**❌ لم يتم تثبيت المتطلبات**
```powershell

```

**❌ المنفذ 5173 مشغول**
```powershell

```
